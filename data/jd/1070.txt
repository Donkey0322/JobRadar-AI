Software Engineering Intern (UNDERGRAD ONLY - distributed systems, public cloud platforms, AI coding tools))
Engineering
San Jose,
California

Apply
Hungry, Humble, Honest, with Heart. The OpportunityAre you a motivated student with solid programming skills, a passion for cloud technologies, and a desire to work in a fast-paced, agile environment? If so, joining the Nutanix Cloud Clusters (NC2) team as a Software Engineer Intern offers you the unique opportunity to gain hands-on experience on a mission-critical public cloud product while contributing to innovative solutions that simplify enterprise AI deployment and enable seamless cloud migration. About the TeamThe Nutanix Cloud Clusters (NC2) team, which helps users run a Nutanix solution in the public cloud by using the same Nutanix software stack used in on-premises environments. The team provides a solution that allows the entire stack to run in the public cloud. NC2 is one of the core products offered by Nutanix, a company recognized as a Leader in Distributed Hybrid Infrastructure and Multicloud Container Platforms. The team's goal aligns with Nutanix's broader mission to enable Hybrid Cloud Adoption and provide seamless migration to the cloud without costly refactoring. Your RoleCollaborate with the Nutanix Cloud Clusters (NC2) team to complete a defined software engineering project.Utilize programming skills, particularly Python, to develop and implement features within a large, distributed codebase.Engage with assigned mentors for guidance and to enhance project development efficiency.Present completed project work and results to the broader NC2 team at the end of the internship.Participate in weekly check-in meetings to discuss progress and address any development roadblocks.Gain hands-on experience with public cloud services (AWS, GCP, Azure) and AI coding assistance tools, if applicable.Document project findings and contributions, showcasing the application of core programming skills in a corporate setting.Contribute to the evolution of seamless cloud migration tools and advancements in enterprise AI deployment capabilities. What You Will BringApplicants must be authorized to work in the US without requiring current or future sponsorship for an employment visa status, such as an F1, H1B, or TN visa.Pursuing a bachelor's degree in Computer Science, Software Engineering, or related fields.Proficiency in programming languages, particularly PythonFamiliarity with public cloud platforms such as AWS, GCP, or Azure (highly desirable).Understanding of AI coding assistance tools like Cursor (considered a plus).Strong problem-solving skills and ability to work independently on defined projects.Effective communication skills for presenting project work to the team.Ability to engage actively with mentorship and participate in regular check-ins.Experience with software development in corporate settings is preferred but not mandatory.Perks You'll Enjoy A structured Campus to Career program with training and mentorship, a hackathon, leadership roundtable discussions, and the chance to showcase your achievements at the close of your internshipPlentiful opportunities for learning, collaboration, and connection in personA detailed look inside our technical Nutanix platform architectureCompetitive pay, Wellness Days, and a vibrant Employee Resource Group communityThe Process We understand you are pursuing multiple employment options. Please note these important steps. Application Screening: We will review your application to understand your unique talents and skills in relation to our opportunities Online Technical Assessment: We will review your foundational skills through a coding skills assessment in the language of your choiceTechnical interviews with our hiring teams to showcase your skills Students who advance to this round will receive preparation prior to the interviews Expect up to 3 interviews with Nutanix engineers and leaders. These are designed to assess your technical skills, role readiness, and your motivation for the opportunity at Nutanix. Each meeting will give you an opportunity to ask questions and learn more about the team you might work with. Work ArrangementHybrid: This role operates in a hybrid capacity, blending the benefits of remote work with the advantages of in-person collaboration. In locations where our workplace policy applies (i.e. San Jose, Durham, Mexico City, Bangalore, Pune, Hoofddorp, Belgrade, Barcelona, Singapore, Sydney and Tokyo), employees are expected to work onsite a minimum of 3 days per week to foster collaboration, team alignment, and access to in-office resources.Workplace type may vary based on location and team requirements. Please speak with your recruiter for details. Additional team-specific guidance and norms will be provided by your manager. var lastScrollTop = 0;
$(window).scroll(function (event) {
var st = $(this).scrollTop();
if (st > 0) {
if (st > lastScrollTop) {
// downscroll code
$('header').addClass('nav-down');
$('header').addClass('smaller')
} else {
// uproll code
$('header').removeClass('smaller');
$('header').removeClass('nav-down');
$('header').addClass('nav-up')
}
} else {
$('header').removeClass('nav-up');
$('header').removeClass('smaller');
}
lastScrollTop = st;
});
jQuery(document).ready(function() {
var offset = 250;
var duration = 300;
jQuery(window).scroll(function() {
if (jQuery(this).scrollTop() > offset) {
jQuery('.back-to-top').fadeIn(duration);
} else {
jQuery('.back-to-top').fadeOut(duration);
}
});
jQuery('.back-to-top').click(function(event) {
event.preventDefault();
jQuery('html, body').animate({scrollTop: 0}, duration);
return false;
})
});
