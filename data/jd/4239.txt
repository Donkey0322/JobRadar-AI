You will contribute to the development and maintenance of scalable systems that process billions of TV ad impressions daily, gaining hands-on experience across backend services, data pipelines, and cloud infrastructure.

● Use AI-ML code agents (for example, Kiro, Copilot, Claude, Cursor, or similar) to design, build, and migrate software systems
● Take ownership of clearly scoped projects, delivering working, high-quality solutions

● Contribute to migrating or modernizing existing services, pipelines, or tools using AI-driven approaches
● Write, test, and iterate on code with a strong emphasis on quality, maintainability, and performance
● Participate in code reviews and incorporate feedback to improve both code and approach
● Document your work, including:
○ What you built
○ How AI was used
