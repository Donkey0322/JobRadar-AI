As a Software Engineering Intern, you will work closely with experienced engineers and product teams to design, build, test, and improve real software used by customers using modern software development tools. Work on meaningful projects.
Learn how large-scale software systems are designed and built
Participate in agile ceremonies and engineering best practices

Write clean, readable, and well-documented code
Participate in code reviews, learning how to give and receive constructive feedback

Debug and fix issues in existing codebases
Contribute to improving code quality, performance, and reliability
Follow secure coding, quality, and engineering best practices
Participate in agile ceremonies (standups, sprint planning, retrospectives)
Document your work and share learnings with the team

Pursuing a Bachelor or Master’s degree in Computer Science, Software Engineering or related fields.
Have basic understanding of software development
Experience with programming languages, specifically Java, Python, Bash scripts, Oracle
Understand fundamental concepts such as data structures, algorithms, and object-oriented programming
Familiarity with AI-assisted coding tools (e.g., GitHub Copilot) (Nice to have)
Strong problem-solving skills and ability to learn quickly.
Have a passion for building responsive and engaging experiences for users
You write high quality, well tested code to meet the needs of your users
You're passionate about building technology at scale and solving challenges for your customers in creative and innovative ways
