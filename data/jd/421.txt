Software Development Intern
Software Development Intern
Lincoln, Nebraska, USA
As a Software Developer Intern at Sandhills Global you will have the opportunity to learn the different modern tools and technologies we use to develop industry-leading software. Each intern is placed in our developer training department where you will go through a series of tasks designed to get you familiar with our processes, procedures and tech stack that we use. Many applications within the company are web-based data driven applications that utilize .NET, SQL Server, Web APIs, and front end frameworks such as javascript, jQuery, and React.js to name a few. Main responsibilities include to maintain, upgrade and create new software for business needs. This could include front end web development, back end business logic changes or database work.
Once through training, Software Developer Interns are placed within an agile team in an area that fits their skills and where we have a need, at which point they can put their new training and skills to work. As part of that team you will have the opportunity to be a part of a highly collaborative team that utilizes agile practices to develop real world software, whether its for improving operational efficiency for the company, developing services we sell to customers, or working on public websites that millions of users visit every month.
We are primarily a Microsoft shop, and utilize technologies in that ecosystem to solve business problems. C# is our primary coding language and what we use for any new projects, but we have many mature systems that are written in VB which we maintain. There are smaller areas that utilize other languages and technology such as PHP, Swift, Objective C, and Java for example, but these are project specific and the minority. Previous experience with Docker, Jenkins, Python can be helpful.
Assume other duties as assigned.This internship is paid at $25/hour!
Internships at Sandhills Global can start at any point in the semester - we are having new hires start their journey with us every single week! Pick a start date that works for you.
•Ability to commute when scheduled to the local office is required•Excellent oral and written communication skills • Able to work independently and communicate within a team environment•Results oriented•Flexible and open to change•Stable and progressive work history•Professional appearance and demeanor

function GetPageURL(decodeURL) {

function openLinkInDialog(url) {
var width = 1000;
var height = 600;
var left = (window.innerWidth - width) / 2;
var top = (window.innerHeight - height) / 2;
var options = "width=" + width + ",height=" + height + ",left=" + left + ",top=" + top;
window.open(url, "_blank", options);
};
var handleKeyPress = function (ModalName) {
return function handleKey(e) {
if (e.keyCode === 9) {
let focusable = document.querySelector(ModalName).querySelectorAll('input, select, textarea, button');
if (focusable.length) {
let first = focusable[0];
let last = focusable[focusable.length - 1];
let shift = e.shiftKey;
if (shift) {
if (e.target === first) { // shift-tab pressed on first input in dialog
last.focus();
e.preventDefault();
}
} else {
if (e.target === last) { // tab pressed on last input in dialog
first.focus();
e.preventDefault();
}
}
}
}
//if user presses esc key then exit modal
if (e.keyCode == 27) {
$("#share-modal").fadeOut(200);
}
}
}
